data types in javascript

there are tow types of data types in javascript

  1. primitive (value types)

  2. Non-Primitive (Reference types)

in this section we will learn about primitive data types

we can classify primitive data types in 7 categorizes

  1. number
  2. string
  3. boolean
  4. undefined
  5. null
  6. bigint
  7. symbol

there is only one type of non primitive data type that is called object

object are further divided into three categorizes

  • object literals
  • array
  • function
